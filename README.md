bmp2txt
=====

### Function
- Read 1, 4, 8 or 24bit's .bmpfile.
- Show datas of bmpfliehead and bmpfliehead in the terminal.
- Output all of the bmpflie's datas in a .txtfile.

### Dependencies
- C++ language
- Linux
- vim editor

### Running
```
make
./b2t bmpname.bmp txtname.txt
vim txtname.txt
:set tabstop=16
```

# bmp2txt

## Function
This program can read 1, 4, 8 or 24bit's .bmpfile,
***
show datas of bmpfliehead and bmpfliehead in the terminal,
***
output all of the bmpflie's datas in a .txtfile.

## Dependencies
C++ language
***
Linux
***
vim editor

## Running
```
make
./b2t bmpname.bmp txtname.txt
vim txtname.txt
:set tabstop=16
```
